# バイターにも品質を！

Quality for Biters! は、進化度に応じて敵ユニットの品質を変化させる Mod です。

ゲーム序盤への影響は控えめですが、
進行するにつれて高品質な敵が徐々に出現するようになります。

## 対象範囲
- ナウビス：
  - すべての敵ユニット
  - バイターの巣
  - スピッターの巣
- グレバ：
  - すべての敵ユニット
  - グレバの巣

## 設計意図
本 Mod は、序盤のバランスを大きく崩すことを目的としていません。
終盤に向かって戦闘圧力を高め、防衛設計の重要性を増すことを狙っています。

「死の世界」「マラソン」設定では、
防衛線の構築と維持がより重要になります。

## 依存関係
- Manis_lib
- Quality Mod（必須）
