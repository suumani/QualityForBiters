# Quality for Biters! — 仕様書（spec）

本書は Quality for Biters! の **設計意図・仕様・合意事項** を記録するためのドキュメントである。
README や Mod Portal では省略される判断基準と、仕様の優先順位を保持することを目的とする。

---

## 0. 目的と非目的

### 0.1 目的
- 敵ユニットに「品質」という成長軸を追加する
- 進化度の上昇に応じて、段階的に戦闘難易度を高める
- 終盤の戦闘に多様性と圧力を加える

### 0.2 非目的
- 序盤の難易度を大きく上げることは目的としない
- 敵の行動 AI を変更することは目的としない
- 特定の惑星のみを極端に強化することは意図しない

---

## 1. 対象範囲

- ナウビス：
  - すべての敵ユニット
  - バイターの巣
  - スピッターの巣
- グレバ：
  - すべての敵ユニット
  - グレバの巣

---

## 2. 品質判定

- 敵の品質は、現在の進化度を基準に確率的に決定される
- 進化度が低い段階では高品質個体は稀
- 進化度が高くなるにつれて、高品質個体の出現率が上昇する

---

## 3. 難易度設計

- ノーマル設定では、難易度上昇は緩やか
- Death World / Marathon 設定では、防衛負荷が大きく増す
- 本 Mod は「持続的な圧力」を目的とし、瞬間的な詰みを狙わない

---

## 4. 互換性と前提

- 本 Mod は Manis_lib に依存する
- Quality Mod が有効であることを前提とする
- グローバルな敵挙動や AI は変更しない

---

## 5. 決定性とマルチプレイ

- 乱数は deterministic な手法を用いる
- マルチプレイでの再現性を前提とする
- 再現性に問題があった場合はバグとして扱う

---

## 6. 本 spec の扱い

- 本書は README および Mod Portal の記述より優先される
- 実装と乖離が生じた場合は、実装を仕様とするか spec を更新するかを明示的に判断する
